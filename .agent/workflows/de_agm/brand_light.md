# Brand Identity - Light Mode

**Aesthetic**: "Clean-Tech", Crisp, Airy, Modern.

## 1. Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| **Background Deep** | Ice White | `#f0f4f8` | Main page background |
| **Background Card** | Pure White Glass | `#ffffff` | Cards, Header (with blur) |
| **Primary Accent** | Deep Cyan | `#00aec1` | Buttons, Icons, Highlights (Darker than neon for contrast) |
| **Secondary Accent** | Royal Blue | `#2962ff` | Gradients, Secondary links |
| **Text Primary** | Dark Jungle | `#1a262b` | Headings |
| **Text Secondary** | Slate Gray | `#546e7a` | Body text |

## 2. Typography
(Shared with Dark Mode)
- **Font Family**: Modern Sans-Serif (e.g., 'Inter', 'Outfit', or system-ui).
- **Headings**: Clean, bold, slight letter-spacing.
- **Body**: Readable, good contrast.

## 3. UI Elements

### Buttons
- **Primary**: Deep Cyan background, White text, Crisp shadow.
- **Ghost**: Outline Deep Cyan, Transparent background.

### Cards
- **Style**: Glassmorphism (Light).
- **Background**: `rgba(255, 255, 255, 0.7)`
- **Border**: `1px solid rgba(0, 174, 193, 0.2)`
- **Shadow**: Soft drop shadow `0 4px 20px rgba(0,0,0,0.05)`.
- **Backdrop**: `blur(12px)`

### Icons
- Thin line style (outline).
- Color: Primary Accent (Deep Cyan).
